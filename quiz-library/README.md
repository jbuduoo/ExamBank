# 題庫下載網站題庫目錄

## 用途

此目錄存放**題庫下載網站**提供的題庫檔案。

這些題庫可以透過 `http://localhost:3000` 下載到 Web 版和 Android 版 App 中。

## 與 App 內建題庫的區別

| 項目 | 題庫下載網站 | App 內建題庫 |
|------|------------|------------|
| **位置** | `data/quiz-library/` | `assets/data/questions/` |
| **用途** | 供用戶下載的題庫 | App 內建的題庫 |
| **讀取方式** | 本地伺服器掃描 | App 打包時包含 |
| **更新方式** | 手動放入檔案 | 需要重新打包 App |

## 使用方式

1. **將題庫檔案放入此目錄**
   - 支援 `.json` 格式
   - 支援多層資料夾結構
   - 檔案會自動出現在下載網站上

2. **啟動本地伺服器**
   ```bash
   npm run quiz-server
   ```

3. **在 App 中下載**
   - Web 版：訪問 `http://localhost:3000`
   - Android 版：輸入電腦的 IP 位址（例如：`http://192.168.1.100:3000`）

## 檔案格式

題庫檔案應符合以下格式：

```json
{
  "source": "題庫來源名稱",
  "importDate": "2025-01-01",
  "questions": [
    {
      "Id": "1",
      "Q": "題目內容",
      "A": "選項 A",
      "B": "選項 B",
      "C": "選項 C",
      "D": "選項 D",
      "Ans": "A",
      "Exp": "詳解內容"
    }
  ]
}
```

## 注意事項

- 此目錄的檔案**不會**被打包到 App 中
- 此目錄的檔案**不會**影響 App 內建的題庫
- 用戶需要透過下載功能才能將題庫匯入到 App
